# PHP Learning 

This repository contains my personal learning journey with the PHP programming language. It includes practice programs, examples, and small projects created while exploring PHP fundamentals and core concepts.

##  Purpose

The goal of this repository is to:

- Learn PHP syntax and programming concepts
- Practice writing clean and reusable PHP code
- Understand server-side scripting
- Build a strong foundation before working with frameworks like Laravel
- Track my progress throughout the learning process

## Topics Covered

This repository includes examples and exercises on:

-  PHP Syntax
-  Variables and Data Types
-  Operators
-  Conditional Statements (`if`, `else`, `switch`)
-  Loops (`for`, `while`, `foreach`)
-  Functions
-  Arrays
-  Strings
-  Forms (GET & POST)
-  Classes and Objects (OOP)
-  File Inclusion (`include`, `require`)
-  Simple Calculator
-  Practice Programs


##  Getting Started

### Prerequisites

- PHP 8.x or later
- XAMPP / WAMP / Laragon (or any PHP server)
- VS Code (Recommended)

### Running the Project

1. Clone the repository

```bash
git clone https://github.com/itsmeumaima/PHP_Learning.git
```

2. Move the project to your server directory

Example (XAMPP):

```
C:\xampp\htdocs\
```

3. Start **Apache** from XAMPP.

4. Open your browser and visit:

```
http://localhost/PHP_Learning/
```

Or open any specific PHP file:

```
http://localhost/PHP_Learning/filename.php
```
